## Welcome

This course package contains everything you need to learn Claude Code - from messy inputs to polished outputs. No coding experience required.

## Getting Started

### Step 1: Prerequisites
- Claude Code installed
- Terminal access
- Mac or Windows computer
- Internet connection
- ~50 minutes per lab

### Step 2: Setup
1. Extract this zip file (if zipped)
2. Open Terminal, navigate to folder
3. Launch Claude Code: `claude .`

### Step 3: Begin Learning
```
/CC.m1.lb1
```

## Case Study: GrowthPath

Throughout this course, you'll work with GrowthPath, a consultancy/professional services company - an early-stage Scottish startup drowning in document chaos, transforming to AI-powered efficiency. Each lab builds on the previous, following Hodl, Flora, Angus, and Isla as they tackle real challenges.
